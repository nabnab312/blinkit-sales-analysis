# Blinkit Sales Analysis

## Project Overview
This project explores Blinkit sales data using Python to identify product performance, customer purchasing behavior, and sales patterns.

## Objectives
- Analyze sales performance
- Identify top-selling products
- Explore category distribution
- Understand customer buying trends

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Workflow
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Data Visualization
5. Insights & Conclusion

## Files
- `Main.ipynb` → Main analysis notebook
- `blinkit_data.csv` → Dataset
